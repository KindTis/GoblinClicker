# 상위 상점 아이템 5종 스펙

작성일: 2026-06-18

## 1. 목적

이 문서는 기존 구매 상점 4종 이후에 배치할 상위 상점 아이템 5종을 정의한다.

목표는 다음과 같다.

- 기존 `녹슨 동전` 단일 재화 구조를 유지한다.
- 기존 업그레이드처럼 레벨 0에서 시작하고 1레벨 단위로 반복 구매한다.
- 기존 비용 공식 `floor(기본 비용 * 성장률 ^ 현재 레벨)`을 유지한다.
- 10분 MVP 초반 밸런스는 깨지 않고, 10분 이후 플레이에 장기 목표를 추가한다.
- 직접 클릭이 가장 빠른 진행 방법이고 자동 피해는 보조 수단이라는 기존 의도를 유지한다.

## 2. 범위

이 스펙은 상위 아이템의 규칙과 구현 기준을 정의한다. 기존 MVP 상점 4종의 기본 역할, 구매 방식, 모바일 상점 동작, 비용 부족 피드백, 처치 전환 중 구매 차단 규칙은 유지한다.

이 스펙에 포함하는 범위는 다음과 같다.

- 신규 업그레이드 ID, 표시명, 비용, 성장률, 효과 공식.
- 기존 피해, 보상, 진흙 함정 계산에 신규 업그레이드를 합성하는 순서.
- 저장 데이터와 UI 표시가 신규 업그레이드 5종을 다루기 위한 요구사항.
- 10분 MVP 밸런스를 보호하기 위한 도달 시점과 검증 기준.

이 스펙에 포함하지 않는 범위는 다음과 같다.

- 신규 재화.
- 업그레이드 잠금 해제 조건.
- x10 구매, 최대 구매, 구매 배수 선택.
- 업그레이드 되팔기와 초기화.
- 전용 튜토리얼 팝업.

## 3. 전제

현재 상점 아이템은 아래 4종이다.

| 표시 순서 | 이름 | 기본 비용 | 성장률 | 효과 |
| ---: | --- | ---: | ---: | --- |
| 1 | 낡은 몽둥이 | 3 | 1.45 | 클릭 피해량 = 1 + 레벨 |
| 2 | 삐걱대는 투석기 | 12 | 1.55 | 5초마다 현재 클릭 피해량 * (1 + 레벨) 자동 피해 |
| 3 | 고블린 미끼 주머니 | 20 | 1.60 | 처치 보상 = 기본 처치 보상 + 레벨 |
| 4 | 진흙 함정 | 90 | 1.85 | 새 고블린 첫 직접 공격 배율 = 1 + 2 * 준비 레벨 |

상위 아이템은 이 4종 아래에 고정 순서로 표시한다. 구매 가능 여부, 추천 상태, 비용 부족 상태가 바뀌어도 행 순서는 바꾸지 않는다.

## 5. 진행 도달 시점

상위 아이템은 기존 10분 MVP 검증 구간을 대체하지 않는다. 10분 active baseline에서는 신규 아이템 구매가 발생하지 않아야 하며, 신규 아이템이 구매 가능 상태에 도달하지 않는 것을 목표로 한다.

기준 도달 시점은 다음과 같다.

| 항목 | 목표 |
| --- | --- |
| 5분 active baseline | 기존 목표인 25~40마리 처치와 2종 이상 기존 업그레이드 경험 유지 |
| 10분 active baseline | 신규 아이템 구매 0회, 신규 아이템 구매 가능 상태 0회 유지 |
| 첫 상위 아이템 목표 | active 기준 10분 이후 첫 장기 목표로 `날 선 전투 도끼` 제시 |
| 첫 상위 아이템 구매 시점 | 상위 아이템을 의도적으로 목표로 삼는 플레이에서 12~18분 사이 |

구현 후 시뮬레이션에서 10분 active baseline의 보유 재화가 `날 선 전투 도끼` 첫 비용 이상이 되면 기본 비용을 올린다. 18분이 지나도 의도적으로 `날 선 전투 도끼`를 목표로 삼은 active 플레이에서 첫 구매가 어렵다면 기본 비용을 내린다. 효과 공식은 비용 조정 뒤에도 유지한다.

## 6. 계산 공식

### 6.1 비용

모든 신규 아이템은 기존 비용 공식을 그대로 따른다.

```text
업그레이드 비용 = floor(기본 비용 * 성장률 ^ 현재 레벨)
```

`현재 레벨`은 구매 전 레벨이다. 모든 신규 아이템의 첫 구매 비용은 기본 비용과 같다.

### 6.2 피해 계산 순서

피해 계산은 다음 순서로 처리한다.

1. 기본 클릭 피해량을 계산한다.
2. 직접 공격이면 진흙 함정 첫 타격 배율을 적용한다.
3. 자동 공격이면 투석기 자동 피해량을 계산한다.
4. `대장장이 계약서` 최종 피해 배율을 적용한다.
5. 최종 피해를 정수로 내림한다.
6. 고블린 체력에서 최종 피해를 차감한다.

최종 피해는 항상 1 이상의 정수여야 한다. 분수 피해와 분수 체력은 만들지 않는다.

### 6.3 클릭 피해량

기본 클릭 피해량은 다음처럼 확장한다.

```text
기본 클릭 피해량 = 1 + 낡은 몽둥이 레벨 + 3 * 날 선 전투 도끼 레벨
```

`날 선 전투 도끼`는 직접 클릭 중심 성장을 강화한다. 이 값은 투석기 자동 피해량의 기반값으로도 사용한다.

### 6.4 투석기 자동 피해량

투석기는 기존처럼 `삐걱대는 투석기` 레벨이 1 이상일 때만 동작한다.

```text
투석기 기본 자동 피해량 =
  기본 클릭 피해량 * (1 + 삐걱대는 투석기 레벨)
  + 기본 클릭 피해량 * 3 * 보강 투석대 레벨
```

`보강 투석대`는 투석기 자동 피해를 강화하지만, 자동 피해 계산이 여전히 클릭 피해량을 기반으로 하므로 직접 공격 업그레이드의 가치가 유지된다.

`보강 투석대`는 `삐걱대는 투석기` 레벨이 0이어도 구매할 수 있다. 다만 투석기 자동 공격은 `삐걱대는 투석기` 레벨이 1 이상일 때만 발생하므로, 투석기 0레벨 상태에서 구매한 `보강 투석대` 효과는 대기 상태로 남는다. 이후 `삐걱대는 투석기`를 구매하면 보유 중인 `보강 투석대` 레벨을 즉시 자동 피해량 계산에 반영한다.

`보강 투석대`를 구매해도 현재 투석기 쿨다운은 리셋하지 않는다. 다음 자동 공격 피해량만 새 레벨 기준으로 계산한다.

### 6.5 처치 보상

처치 보상은 다음처럼 확장한다.

```text
처치 보상 =
  기본 처치 보상
  + 고블린 미끼 주머니 레벨
  + floor(기본 처치 보상 * 0.4 * 황금 미끼 항아리 레벨)
```

`황금 미끼 항아리`는 고블린 레벨이 높아질수록 효율이 커지는 중후반 재화 성장 아이템이다. 초반 고정 보상 증가 역할은 기존 `고블린 미끼 주머니`가 유지한다.

### 6.6 진흙 함정 준비 레벨

새 고블린 등장 시 준비 레벨은 다음처럼 계산한다.

```text
진흙 함정 준비 레벨 =
  진흙 함정 레벨
  + 2 * 깊은 진흙 수렁 레벨
```

첫 직접 공격 배율 공식은 기존과 동일하게 유지한다.

```text
첫 직접 공격 배율 = 1 + 2 * 진흙 함정 준비 레벨
```

전투 중 `깊은 진흙 수렁`을 구매해도 현재 고블린에게 이미 준비된 진흙 함정 값은 바꾸지 않는다. 다음 고블린 등장 시점부터 반영한다.

### 6.7 최종 피해 배율

`대장장이 계약서`는 직접 피해와 자동 피해 계산의 마지막 단계에 적용한다.

```text
최종 피해 = floor(계산된 피해 * (1 + 0.18 * 대장장이 계약서 레벨))
```

직접 공격에서는 진흙 함정 첫 타격 배율까지 적용한 뒤 마지막에 곱한다. 자동 공격에서는 투석기 자동 피해량을 계산한 뒤 마지막에 곱한다.

## 7. 상태와 저장 규칙

신규 아이템 5종은 기존 `upgrades` 레코드에 같은 방식으로 저장한다. 각 값은 0 이상의 안전한 정수 레벨이다.

신규 업그레이드 키는 다음과 같다.

```text
battleAxe
reinforcedCatapult
goldenBaitJar
deepMudBog
blacksmithContract
```

기존 저장 데이터에 신규 키가 없으면 로드 시 해당 레벨을 0으로 정규화한다. 기존 저장 데이터를 깨지 않기 위해, 저장 구조의 의미가 바뀌지 않는다면 저장 버전은 유지할 수 있다.

`mudTrapArmedLevel`은 `진흙 함정 레벨 + 2 * 깊은 진흙 수렁 레벨`까지 허용한다. 저장 복원 시 `mudTrapArmedLevel`이 이 최대 준비 레벨보다 크면 최대 준비 레벨로 낮춘다. `진흙 함정`과 `깊은 진흙 수렁`이 모두 0이면 `mudTrapArmedLevel`은 0으로 정규화한다.

## 8. 상점 표시 규칙

상점 표시 순서는 다음으로 고정한다.

1. 낡은 몽둥이
2. 삐걱대는 투석기
3. 고블린 미끼 주머니
4. 진흙 함정
5. 날 선 전투 도끼
6. 보강 투석대
7. 황금 미끼 항아리
8. 깊은 진흙 수렁
9. 대장장이 계약서

신규 아이템은 숨기거나 별도 잠금 상태로 두지 않는다. 재화가 부족하면 기존 상점 규칙처럼 비용 부족 상태와 부족한 `녹슨 동전` 수를 보여준다.

`보강 투석대`는 `삐걱대는 투석기` 레벨이 0이어도 숨기거나 구매 불가로 만들지 않는다. 대신 효과 설명에서 투석기 보유 시 자동 피해가 강화된다는 조건을 드러낸다.

신규 아이템은 `낡은 몽둥이`의 첫 구매 추천 표시와 별개로 추천 표시를 갖지 않는다. 모바일 하단 `상점` 버튼의 구매 가능 개수는 기존 4종과 신규 5종을 모두 포함해 계산한다.

각 신규 아이템은 기존 업그레이드 행과 같은 정보 구조를 사용한다.

- 아이콘.
- 이름.
- 현재 레벨.
- 다음 1레벨 비용.
- 다음 효과 설명.
- 구매 가능 여부.
- 비용 부족 상태.

## 9. UI와 피드백 요구사항

신규 아이템 구매 성공, 비용 부족, 처치 전환 중 입력 차단 피드백은 기존 상점 규칙을 따른다.

신규 아이템 5종에는 각각 고유 아이콘 에셋 키를 배정한다. 임시 아이콘을 쓰더라도 `UpgradeId`와 UI 표시 매핑은 누락되면 안 된다.

피해와 보상 표시 문구는 플레이어가 다음 구매 효과를 이해할 수 있어야 한다. 단, 상점 행 안에서 긴 공식을 모두 노출하지 않고 짧은 효과 설명을 우선한다.

예시:

| 이름 | 상점 효과 설명 예시 |
| --- | --- |
| 날 선 전투 도끼 | 클릭 피해 +3 |
| 보강 투석대 | 투석기 보유 시 자동 피해 증가 |
| 황금 미끼 항아리 | 처치 보상 비례 증가 |
| 깊은 진흙 수렁 | 다음 고블린 첫 타격 강화 |
| 대장장이 계약서 | 모든 피해 최종 배율 증가 |

## 10. 밸런스 의도

`날 선 전투 도끼`는 가장 먼저 도달하는 상위 목표다. 기존 클릭 성장의 흐름을 끊지 않고, 플레이어가 직접 클릭으로 더 빠르게 진행할 이유를 만든다.

`보강 투석대`는 자동 피해 강화지만 클릭 피해량을 기반으로 계산한다. 따라서 투석기만 올리는 전략보다 직접 피해 업그레이드와 함께 올리는 전략이 더 좋다.

`황금 미끼 항아리`는 고블린 레벨이 오른 뒤 가치가 커지는 보상 성장 아이템이다. 기본 처치 보상에 비례하므로 너무 이른 구간에서는 과도한 눈덩이 효과를 만들지 않는다.

`깊은 진흙 수렁`은 기존 진흙 함정의 역할을 확장한다. 새 고블린의 첫 직접 공격을 강하게 만들지만, 전투 중 구매가 현재 고블린에게 즉시 반영되지는 않는다.

`대장장이 계약서`는 장기 플레이용 최상위 피해 배율이다. 첫 구매 비용과 성장률이 높기 때문에 10분 active baseline에는 영향을 주지 않는 것을 목표로 한다.

상위 아이템 비용은 효과 공식보다 우선해서 조정하는 밸런스 손잡이다. 도달 시점이 너무 빠르거나 늦으면 기본 비용과 성장률을 조정하고, 아이템 역할과 효과 공식은 유지한다.

## 11. 검증 기준

구현 시 최소 검증 기준은 다음과 같다.

- 신규 아이템 5종이 기존 4종 뒤에 고정 순서로 표시된다.
- 각 신규 아이템의 첫 구매 비용이 기본 비용과 일치한다.
- 각 신규 아이템의 다음 비용이 `floor(기본 비용 * 성장률 ^ 현재 레벨)`로 계산된다.
- `날 선 전투 도끼` 구매 후 직접 클릭 피해량이 증가한다.
- `보강 투석대` 구매 후 투석기 자동 피해량이 증가하고, 투석기 쿨다운은 리셋되지 않는다.
- `삐걱대는 투석기` 레벨 0에서 `보강 투석대`를 구매하면 자동 공격은 새로 발생하지 않고, 이후 `삐걱대는 투석기` 첫 구매 시 보유 중인 `보강 투석대` 레벨이 자동 피해량에 반영된다.
- `황금 미끼 항아리` 구매 후 처치 보상이 증가한다.
- `깊은 진흙 수렁` 구매 효과는 다음 고블린 등장 시 준비 레벨에 반영된다.
- `대장장이 계약서` 구매 후 직접 피해와 자동 피해의 최종값이 증가한다.
- 기존 저장 데이터를 로드하면 신규 업그레이드 5종 레벨이 0으로 정규화된다.
- `mudTrapArmedLevel` 저장 복원 상한이 `진흙 함정 레벨 + 2 * 깊은 진흙 수렁 레벨`을 사용한다.
- 모바일 하단 `상점` 버튼의 구매 가능 개수는 신규 아이템 구매 가능 상태를 포함한다.
- 5분 active baseline은 기존 목표인 25~40마리 처치와 2종 이상 업그레이드 경험을 유지한다.
- 10분 active baseline은 기존 목표인 20마리 이상 처치, `낡은 몽둥이` 구매, 3회 이상 구매를 유지한다.
- 10분 active baseline에서 신규 아이템 구매 횟수는 0회다.
- 10분 active baseline에서 보유 재화는 `날 선 전투 도끼` 첫 구매 비용보다 낮다.
- 10분 active baseline에서 모바일 하단 `상점` 버튼의 구매 가능 개수에 신규 아이템은 포함되지 않는다.
- 상위 아이템을 의도적으로 목표로 삼는 active 플레이에서 `날 선 전투 도끼` 첫 구매는 12~18분 사이에 발생한다.

## 12. 후속 검토 항목

다음 항목은 구현 후 플레이 테스트와 시뮬레이션 결과를 보고 조정한다.

- `날 선 전투 도끼` 첫 구매 목표 시간 12~18분이 실제 플레이 감각에 맞는지 확인한다.
- `황금 미끼 항아리`가 장기 경제를 과도하게 가속하지 않는지 확인한다.
- `대장장이 계약서`의 최종 피해 배율이 높은 레벨에서 고블린 체력 성장률을 압도하지 않는지 확인한다.
- 모바일 상점에 9개 아이템이 모두 표시될 때 스크롤, 포커스, 하단 토글 가림 문제가 없는지 확인한다.
